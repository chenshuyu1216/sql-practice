## Current Progress

I have learned the following SQL and database basics:

1. Tables
   - A table stores one type of structured data.
   - Rows are records, and columns are fields.

2. Keys
   - A primary key uniquely identifies each row.
   - A foreign key connects data between different tables.

3. Creating databases
   - A database stores related tables and data.
   - CREATE DATABASE is used to create a new database.

4. Storing data
   - Data is stored in tables.
   - INSERT INTO is used to add records.

5. Constraints
   - Constraints control what data can be stored.
   - Examples include NOT NULL, UNIQUE, PRIMARY KEY, and FOREIGN KEY.

6. Updating and deleting data
   - UPDATE changes existing records.
   - DELETE removes records.
   - WHERE is important because it limits which rows are changed or deleted.
