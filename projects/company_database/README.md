# Company Database SQL Project

This is a beginner SQL mini project based on a simple company database.

## Project Overview

The database contains four tables:

- `employee`
- `branch`
- `client`
- `works_with`

This project practices basic SQL database design and query writing.

## What I Practiced

- Creating tables
- Primary keys
- Foreign keys
- Insert data
- Update data
- Basic SELECT queries
- ORDER BY and LIMIT
- Aggregate functions
- JOIN
- LEFT JOIN
- Subqueries
- GROUP BY
- HAVING

## Files

```text
company_database/
├── schema.sql      # Table structure
├── seed.sql        # Sample data
├── queries.sql     # Practice queries
└── README.md       # Project description
```

## How To Use
Run the files in this order:

1. `schema.sql`
2. `seed.sql`
3. `queries.sql`
## What I Learned
I learned how different tables are connected using foreign keys.
I also learned that JOIN is used to combine related tables, while GROUP BY and HAVING are used to summarize and filter grouped results.
